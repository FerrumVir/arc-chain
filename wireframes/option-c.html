Technology

Purpose-built from scratch

Every layer of the stack is designed for AI workloads. No forks, no shortcuts.

DAG Consensus

VRF-based proposer selection with stake-weighted leader rotation. Deterministic finality in ~4 seconds.

Block-STM Execution

Parallel transaction execution with optimistic concurrency. Abort-and-retry on conflicts.

Stwo STARK Proofs

Circle STARK over M31 field. 22-constraint AIR with inner-circuit recursion for composable proof aggregation.

GPU Verification

Metal + WGSL shaders for Ed25519 batch verification. Branchless Shamir trick with zero SIMD divergence.

QUIC Transport

Reed-Solomon FEC erasure coding with 50% redundancy. PEX peer discovery for sub-second block propagation.

Jellyfish Merkle Tree

Sharded state with incremental updates. WAL persistence and cross-shard locking for atomic operations.
